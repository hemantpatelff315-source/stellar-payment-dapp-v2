# 🚀 Stellar Payment dApp

A modern Stellar Testnet Payment dApp built with Vite, JavaScript, Stellar SDK, and Freighter Wallet.

## 🌐 Live Demo

https://stellar-payment-dapp-v2-tau.vercel.app

## ✨ Features

- 🔗 Connect Freighter Wallet
- 💰 View Live XLM Balance
- 📤 Send XLM on Stellar Testnet
- 📋 Copy Wallet Address
- 📱 QR Code for Wallet Address
- 🧾 Transaction History
- 🔐 Secure Transaction Signing
- ⚡ Responsive UI
- 🌙 Modern Dashboard

## 🛠 Tech Stack

- JavaScript
- Vite
- Stellar SDK
- Freighter API
- HTML5
- CSS3

## 🚀 Installation

```bash
git clone https://github.com/hemantpatelff315-source/stellar-payment-dapp-v2.git

cd stellar-payment-dapp-v2

npm install

npm run dev
```
